# v0.7.1 Release Plan

**Target date:** April 2026
**Scope:** Packaging of post-0.7.0 dungeon-editor parity work and the welcome
screen overhaul. No new cross-cutting initiatives.

---

## Release themes

1. **Welcome screen / project startup UX** — single guided New Project modal,
   async ROM metadata scanning, recent-projects undo + pin, command-palette
   surface for welcome actions.
2. **Dungeon editor parity and polish** — BG1/BG2 layout routing, pit-mask
   correctness, replay-geometry selection, ROM-backed parity tests, leaner
   workbench navigation, lazy room materialization.
3. **Editor memory/startup footprint** — session editors constructed lazily,
   hidden full-mode asset loads deferred, overworld eager bitmap trim,
   render-target texture split.
