Detailed Description

Manual object renderer for debugging and testing basic object rendering.

This class provides simple, manual rendering of basic dungeon objects to help debug the graphics pipeline and understand object data structures.

Features:

  • Manual tile creation for simple objects
  • Direct graphics sheet access
  • Palette debugging and testing
  • Simple pattern rendering (solid blocks, lines, etc.)
