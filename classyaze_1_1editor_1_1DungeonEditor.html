Detailed Description

DungeonEditor class for editing dungeons.

This class provides a comprehensive dungeon editing interface that integrates with the new unified dungeon editing system. It includes object editing with scroll wheel support, sprite management, item placement, entrance/exit editing, and advanced dungeon features.
